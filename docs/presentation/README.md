# RepoQuest Presentation Pack

This folder contains short, presentation-ready documents for turning RepoQuest into a hackathon pitch deck, demo script, or judging handout.

## Recommended Use

Use these files in this order:

1. `one_pager.md` - Fast summary for judges, teammates, or submission forms.
2. `slide_deck_outline.md` - Copy into PowerPoint, Google Slides, Canva, or Gamma.
3. `demo_talk_track.md` - Practice the 90-second live demo.
4. `ai_fusion_story.md` - Explain the new AI-first hybrid analyzer clearly.
5. `judge_qna.md` - Prepare answers for predictable judge questions.

## Core Message

RepoQuest turns an unfamiliar repo into an onboarding quest.

It scans a small codebase safely, maps what it finds, creates a reading path, generates component cards, builds a quiz, exports an onboarding guide, and optionally uses AI Fusion to audit and improve the deterministic interpretation.

## Positioning

RepoQuest is not a generic repo chatbot. It is a codebase onboarding analyzer:

- deterministic evidence first
- optional AI interpretation second
- no uploaded code execution
- no dependency installation
- no credentials required for the default demo
- Streamlit Cloud friendly
